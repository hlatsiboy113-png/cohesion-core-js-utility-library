# Cohesion Core JavaScript Utility Library

A lightweight collection of reusable JavaScript utility functions designed to simplify common data transformation tasks such as string manipulation, validation, and counting operations.

---

##  Utilities Included

- slugify
- countWords
- capitalize
- truncateText
- reverseString
- deepClone
- formatDate
- formatTime
- isPalindrome
- randomInt
- removeSpaces
- uniqueArray

---

##  Features

- Pure JavaScript (no dependencies)
- ES Module based (`export / import`)
- Well-documented functions
- Built-in test cases using `console.assert`
- Handles edge cases (invalid input, empty values, etc.)

---
